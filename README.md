# build-agents

Starter marketing site and planning docs for **Build Agents** — a no-code AI agent builder for non-technical users.

## Files
- `PRD.md` — product requirements for v1
- `landing-copy.md` — conversion-focused homepage copy
- `index.html` + `styles.css` — static landing page scaffold

## Run locally
Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8080
```

Then visit <http://localhost:8080>.
