# Build Agents — Brand Kit v1

## Brand Position
Build Agents helps non-technical people create production-ready AI agents using plain language.

## Brand Personality
- Clear
- Calm
- Competent
- Modern
- Human-first

## Voice
- Direct, simple, non-jargon
- Confident, not hypey
- Helpful and practical

## Core Colors
- `Midnight` — `#0B1020` (primary dark)
- `Electric Blue` — `#3B82F6` (primary action)
- `Cyan Glow` — `#06B6D4` (accent)
- `Cloud` — `#F8FAFC` (light background)
- `Slate` — `#334155` (body copy)

## Typography
- Primary: Inter
- Fallback: system-ui, -apple-system, Segoe UI, Roboto, Arial

## Logo Rules
- Prefer symbol + wordmark on white/light backgrounds
- Use white logo variants on dark/photo backgrounds
- Keep clearspace equal to the icon radius on all sides
- Minimum digital width:
  - Symbol: 24px
  - Wordmark: 120px

## Deliverables
- Logo SVGs (symbol, horizontal, stacked, monochrome)
- Poster SVGs (A4 portrait and social dimensions)
- Social preview card SVG
- Favicon/icon SVG
