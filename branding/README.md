# Branding Assets — Build Agents

This folder contains the initial branding pack for Build Agents.

## Structure
- `docs/BRAND.md` — brand strategy and usage rules
- `logo/` — primary logo variants (SVG)
- `icons/` — favicon/icon assets (SVG)
- `posters/` — launch poster designs (SVG)
- `social/` — Open Graph and social cards (SVG)

## Suggested Next Step
If you want print-ready and social-exported PNG/WebP versions, use Figma/Illustrator or run an SVG export script to generate fixed-size raster outputs.
